一个简单的个人基金计算系统
过于简单就不写过多内容了，感觉还有很多功能可以添加进去，但时间有限，就直接ai生成这篇了


1. **`check_csv()` - 命令行调度与安检**
   负责拦截 `sys.argv` 传参，确保用户传入了正确格式的 `.csv` 数据源。它充当系统的第一道物理防线。
   
2. **`read_csv(funds_csv)` - 本地数据反序列化引擎**
   使用 `csv.DictReader` 将静态的硬盘文本数据清洗、转换为 Python 内存中的字典列表，并极其严谨地将字符串强制转换为 `float` 类型，为后续的数学运算扫清障碍。
   
3. **`update(funds_csv)` - 增删改查 (CRUD) 覆写模块**
   这是一个极其强大的状态机。它允许用户在终端输入代码修改持仓份额。如果遇到全新的基金代码，它会触发 `Upsert` 逻辑，极其智能地将新基金追加进系统，并通过 `csv.DictWriter` 完成对硬盘的底层数据覆写。

4. **`get_funds(code)` - 网络情报特工 (API Fetcher)**
   利用 `requests` 库接入天天基金网的真实公开 API。为了应对反爬策略，该模块使用高级的**字符串切片魔法**剥离了 JSONP 伪装皮，并提取出最新的净值数据。自带 HTTP 状态异常熔断保护。

5. **`calculate(funds_list)` - 核心算盘引擎**
   极其纯粹的数学模块。负责遍历超级字典，结合持仓份额、历史成本和最新净值，精准计算每一笔交易的 P&L（盈亏），并汇总出总仓位的实时资产表现。

### 如何运行
1. 确保安装依赖：`pip install -r requirements.txt`
2. 启动监控终端：`python project.py funds.csv`
3. 根据终端提示，选择是否修改持仓，或直接按 `n` 查阅报表。
